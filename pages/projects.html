Gameuh is a free open-source game built in Python using Pygame only. It is a side-scroller with looter elements inspired by Exile-Likes game such a Path of Exile or Diablo.

The objective of the project is to produce a game that is not only fun to play, but also fully covered in unit tests and modular to allow easy contributions from the open-source community such as mods or forks. The code is also linted for maximum readability.

The ressources used in the game are all either created by me or sourced from free-to-use ressources with proper attributions in the credits section of the game.




LearN 2 SlitheR

TEACHING A SNAKE HOW TO SLITHER



Learn 2 Slither is an AI project built in Python. In order for the snake to survive the longest, it has to learn how to avoid walls and its own tail while going after food.

The AI is built using a Q-Learning algorithm, which is a type of reinforcement learning. The snake is rewarded for eating food and penalized for hitting walls or its own tail. Over time, the AI learns the best actions to take in order to maximize its score.

The project also includes a visualization of the training process, showing how the snake's performance improves over time.

The graphical interface is built using Pygame.
